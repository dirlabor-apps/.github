# Código de conducta

## Propósito
La organización de GitHub de DirLabor busca mantener un entorno de trabajo respetuoso, colaborativo, técnico y seguro para todas las personas participantes.

## Comportamiento esperado
- Mantener una comunicación respetuosa y profesional.
- Aportar críticas técnicas constructivas.
- Documentar decisiones y cambios con claridad.
- Respetar tiempos, acuerdos y lineamientos del equipo.
- Promover la colaboración y el aprendizaje.

## Comportamiento inaceptable
- Lenguaje ofensivo, discriminatorio o intimidante.
- Ataques personales o descalificaciones.
- Publicación de información sensible sin autorización.
- Sabotaje deliberado del trabajo técnico o documental.
- Conductas contrarias al respeto institucional.

## Aplicación
Este código aplica a issues, pull requests, comentarios, documentación, discusiones y demás espacios de colaboración asociados a la organización.

## Reporte
Los incidentes deben reportarse al canal institucional definido por DirLabor para gestión administrativa o técnica correspondiente.

## Medidas
La organización podrá tomar acciones proporcionales, incluyendo advertencias, restricción de participación o retiro de acceso.
